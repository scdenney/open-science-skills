# Fact Check

Fact-check the claims in the manuscript below against their cited sources. First run `citation-check` (in-text/reference parity and source existence). Then, for each in-text citation or footnote that backs a substantive claim, locate the source's knowledge-base Markdown file (`sources/md/` or `knowledge_base/md/`) and verify the source actually supports the claim — flagging contradiction, overclaiming, scope creep, stale numbers, misattribution, and sources missing from the knowledge base. Quote the supporting or contradicting passage for every verdict.
