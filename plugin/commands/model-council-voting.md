# Model Council Voting

Run a contested labeling, scoring, or term-discovery task through a diverse panel of language models as independent coders, then read their (dis)agreement: assemble jurors from different training families to decorrelate errors, keep votes independent (no deliberation), apply a pre-stated k-of-N consensus rule with absolute floors and theory-driven category filters, report chance-corrected agreement (Cohen/Fleiss/Krippendorff) as reliability not validity, address the correlated-errors caveat, and validate against a human sample and an independent non-LLM method.
